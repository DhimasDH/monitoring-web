# monitoring-web
Repository untuk Projek Internet of Things Prodi D4 Teknik Telekomunikasi Politeknik Negeri Semarang

Cara membuka File Program ini sebagai berikut:
- Download semua file program dan database dan masukkan kedalam folder {folder_name}
- Upload database ke MySQL atau yang lainnya
- Buat User Profile untuk terhubung dengan database ( User: user_pendaki & Password: kelompok4)
- Hubungkan antara database dengan User Profile
- buka hasil program tersebut di mesin penelusuran dengan memasukkan localhost/{folder_name}
